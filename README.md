KrishiAI is a comprehensive, full-stack digital solution designed to empower farmers with data-driven insights. By leveraging Artificial Intelligence, the system provides real-time soil analysis, expert agricultural advisory, and localized weather forecasting to help optimize crop yield and sustainable farming practices.
**Tech Stack**

**Frontend**
HTML5 / CSS3 / JavaScript (ES6+): Responsive UI with a focus on ease of use for rural users.
FontAwesome: For intuitive iconography.
Google Fonts (Poppins): For clean and modern typography.

**Backend**
Node.js & Express: Scalable server architecture for handling API requests and file processing.
Multer: Middleware for handling soil image uploads.
Axios: For fetching real-time data from external weather and agricultural APIs.

**AI / ML Integration**
Computer Vision: TensorFlow.js or OpenAI Vision for soil classification.
NLP: Integration with LLMs for the “Ask Advisor” feature.
